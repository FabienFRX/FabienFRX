# 🚀 PoC Ionic React - Application Mobile AREA

## Objectif
Tester Ionic React pour une application mobile hybride : réutilisation du code React, composants natifs, et développement web-first avec Capacitor.

## 🛠️ Installation rapide

```bash
# Aller dans le dossier du PoC
cd poc-mobile-ionic

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Accéder à l'application
# → http://localhost:8100
```

## 📱 Fonctionnalités
- Navigation Ionic Router (React Router)
- Appels API REST avec axios
- Composants Ionic natifs (IonCard, IonList, etc.)
- Pull-to-refresh natif
- Réutilisation 100% du code React
- Hot reload Vite (~2s)
