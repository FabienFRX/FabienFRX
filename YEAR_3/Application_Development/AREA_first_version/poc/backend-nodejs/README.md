# Node.js POC for AREA Project

## Overview
This POC demonstrates Node.js + Express.js capabilities for the AREA project, showcasing event-driven architecture and real-time features essential for automation platforms.

## Features Demonstrated
- ✅ Event-driven architecture with Express.js
- ✅ Real-time capabilities foundation
- ✅ CORS support for web client integration
- ✅ Required `/about.json` endpoint
- ✅ Non-blocking I/O operations
- ✅ Lightweight and fast startup

## Key Advantages
- **Real-time**: Perfect for AREA hooks and live triggers
- **JavaScript Everywhere**: Same language for frontend/backend
- **Event-driven**: Natural fit for automation workflows
- **Large Ecosystem**: Huge npm library selection
- **Quick Development**: Rapid prototyping and iteration

## API Endpoints
- `GET /hello/` - Test endpoint showing Node.js capabilities
- `GET /about.json` - Required project endpoint with services
- `GET /realtime-demo/` - Demonstrates real-time capabilities
- `GET /health` - Health check endpoint

## Quick Start
```bash
cd poc/backend-nodejs
npm install
npm start
```

## Why Node.js for AREA
- Excellent for real-time applications (WebSockets, SSE)
- Event-driven nature matches automation workflows
- Single language across full stack
- Rapid development and prototyping
- Strong ecosystem for OAuth2 integrations
