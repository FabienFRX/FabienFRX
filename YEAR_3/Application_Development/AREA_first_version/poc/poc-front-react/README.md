# 🚀 POC React - Builder d'AREA

## Objectif
Tester React sur le cœur métier de notre plateforme IFTTT : la création et gestion des AREAs (Action + REAction)

## 🛠️ Installation rapide

```bash
# Cloner le repository
git clone [url]
cd poc-front-react

# Installer les dépendances
npm install

# Lancer le POC
npm start
