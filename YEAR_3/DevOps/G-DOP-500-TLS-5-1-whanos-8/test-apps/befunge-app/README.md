# Whanos Befunge Test Application

Simple Befunge program for testing Whanos CI/CD pipeline.

## About

Befunge is a two-dimensional esoteric programming language. This program prints "Hello World!".

## Run

```bash
# You need a Befunge interpreter to run this
befunge app/main.bf
```

## Whanos Deployment

This application is automatically detected by Whanos as a Befunge project (presence of `app/main.bf`).
